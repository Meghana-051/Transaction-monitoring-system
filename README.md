# AI-Based Transaction Monitoring & Fraud Detection System

A full-stack MERN project to manage financial transactions and detect suspicious transactions using rule-based fraud logic.

## Features
- Add, edit, delete transactions
- Fraud detection system
- Fraud highlighting
- Dashboard summary
- Filters and search

## Tech Stack
- React
- Node.js
- Express.js
- MongoDB

## Run Project

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
